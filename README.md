# LoginFlowExpress
This shows a basic Token based login flow using React and Node.

#Server
1. cd server dir
2. Run npm i
3. Run npm start
4. Server is running at port 3001 (localhost)

#Client
1. cd client dir
2. Run npm i
3. Run npm start
4. Launch http://localhost:9001
